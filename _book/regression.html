Part 5 Regression

This chapter shows how to perform linear (5.1) and logistic (5.2) regressions

5.1 Linear regression

  1. To perform a logistic regression you need a continuous predictor variable and a categorical outcome variable. Therefore, your data should look something like this:

  2. If the outcome variable has two levels (as in the above image): select Analyses -> Regression -> 2 Outcomes. If the outcome variable has three or more levels, select Analyses -> Regression -> N Outcomes.

  3. Move your outcome variable to Dependent Variable and your predictor to Covariates.
